Research & Publications

Featured Research · Dissertation Topic

LLM-Based Multimodal & Explainable Epidemic Forecasting

A dual-LLM architecture that fuses numerical time series with expert prior knowledge and surveillance reports given as text, fine-tuned with LoRA — up to 26% improvement over state-of-the-art forecasters and effective under data scarcity.

IEEE Journal of Biomedical and Health Informatics, 2025 (MIFlu)

  • LLM
  • Multimodal
  • LoRA/PEFT
  • Prompting
  • Influenza Forecasting

Epidemic & Health Time-Series Forecasting

Multimodal and interpretable models for influenza / COVID-19 surveillance data.

  • RESEAT: Multi-Regional Time-Series Forecasting

    Recurrent self-attention network where LSTMs learn per-region temporal patterns and recurrently connected attention weights model dynamically changing inter-regional relationships, outperforming SOTA on influenza/COVID-19 forecasting while visualizing region correlations.

    IEEE Journal of Biomedical and Health Informatics, 2023

    • Self-Attention
    • LSTM
    • Multivariate Time Series
    • Interpretability

  • Two-Stage Data Selection for Long-Term Influenza Forecasting

    Automatically selects which other countries’ surveillance data (168 countries, WHO FluNet) help long-term (26-week) influenza prediction, reducing RMSE by up to ~47% versus conventional inputs across 6 ML models and 3 target countries.

    Computers, Materials & Continua, 2021

    • Feature Selection
    • Long-Term Forecasting
    • Machine Learning

Meta-Learning for Data-Scarce Prediction

Few-shot adaptation when target data is limited — forecasting and anomaly detection.

  • MARAPAS: Region-Adaptive Meta-Learning

    Region-adaptive MAML initialization for influenza forecasting (MARAPAS), and TAPT — a task-adaptive parameter transformation that improves few-shot animal sound classification accuracy by up to 20.32% over ProtoNet on public benchmarks.

    Journal of King Saud University – Computer and Information Sciences, 2023

    • MAML
    • Few-Shot Learning
    • Meta-Learning
    • Bioacoustics

  • MAVAE: Meta-Learned VAE for Anomaly Detection

    Unsupervised anomaly detection for facility management that meta-learns shared knowledge from normal data across facilities, adapting to new facilities with few samples — ~45% average improvement over SOTA (F1/AUC) on the SKAB benchmark.

    Journal of Building Engineering, 2023

    • VAE
    • MAML
    • Anomaly Detection
    • Sensor Time Series

Generative Modeling & Data Augmentation

Synthesizing data to train forecasters where records are scarce.

  • GAN-Based Data Augmentation for Load Forecasting

    A two-stage generation scheme — CTGAN synthesizes virtual input variables, then a regression model trained on real data produces the corresponding outputs — that consistently boosts short-term electric load forecasting accuracy in data-scarce settings.

    IEEE Access, 2020

    • CTGAN
    • Data Augmentation
    • Load Forecasting

Projects

  • Explainable ILI Forecasting with LLMs

    First-author research project: combines ILI time series with CDC FluView / Japan IDWR surveillance reports, using GPT-generated explanation labels to LoRA fine-tune LLaMA-3.1-70B (LLaMA-Factory, DeepSpeed ZeRO-3) so the model produces predictions together with the reasoning behind them. Includes an LLM-judge evaluation of explanation quality and classic TSF baselines (DLinear, PatchTST, etc.).

    • LLaMA-3.1-70B
    • LoRA
    • LLaMA-Factory
    • DeepSpeed
    • vLLM

  • Hierarchical AI Agent Framework

    A hierarchical multi-agent chatbot framework built for an Incheon International Airport chatbot project: a main agent recursively delegates tasks to sub-agents (browser, PDF, vector-DB), and every action is emitted as executable Python code (CodeAct/ReAct paradigm). Native functions, MCP server tools, and sub-agents are all exposed to the LLM as uniform callable tools.

    • Multi-Agent
    • CodeAct/ReAct
    • MCP
    • LangChain
    • Python
